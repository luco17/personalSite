---
title: Codex can run two branches at once
date: 2026-05-06
---

I can instruct Codex to run a second branch in a worktree on another localhost port while the main branch runs on the original port.

That makes it easy to compare two versions side by side, especially when a designer or non-technical person has made UI changes I do not want to merge yet.

It is easier than setting up branch previews, and much nicer than repeatedly starting and stopping servers while switching branches.
